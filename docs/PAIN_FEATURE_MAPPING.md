# Pain → Feature Execution Map

هذا المستند يحوّل التحليل السابق إلى باكلوج عملي موزّع حسب الأولوية والجدول الزمني (0–12 شهر). تم اختيار العناصر الأقرب للتنفيذ مع الأخذ بالحسبان ما تم بناؤه بالفعل (Adaptive Learning، Thinking Assistant، Group Mode).

| # | Pain / Opportunity | Feature / Mode | ما الذي سنبنيه؟ | Metrics | Priority |
|---|--------------------|----------------|-----------------|---------|----------|
| P1 | ملل استخدام الهاتف في محتوى بلا قيمة + ضعف التركيز | **Daily Brain Gym** (Solo) | جلسة يومية من 3 مسارات قصيرة بوقت محدد، مع Mind Flex Score يستند إلى Adaptive Learning | D1/D7 retention، متوسط Mind Flex | High (0-3M) |
| P2 | فجوة الأجيال وعدم وجود نشاط عائلي | **Family Link Sessions** | تحسين Group Mode الحالي: ملفات لاعبين عائليين، تحديات أسبوعية، Family Bonding Meter بسيط | Weekly Active Families، معدل إكمال تحديات الأسرة | High (0-3M) |
| P3 | ضعف المرونة المعرفية في التعليم/الشركات | **Classroom / Workshop Pack** (Pilot) | حزم قابلة للمشاركة (offline bundle) + لوحة تقدم بسيطة للمدرّس | # المؤسسات المشاركة، معدل التكرار شهريًا | Medium (3-6M) |
| P4 | الحاجة إلى محتوى محلي عربي/هندي/لاتيني | **Culture Link Packs** | حزم موسمية (أمثال، أحداث، رموز محلية) + Daily World رابط للعالم | % جلسات في المحتوى المحلي، مشتريات الحزم | High (3-6M) |
| P5 | ربط السلوك بنتائج الحياة (طول عمر/ندم/مشاعر) | **Life Journey Modes** (Regret / Emotion / Longevity) | ألغاز محسوبة من 3-4 روابط تشرح السلوك ↔ النتيجة ↔ البديل الصحي، مرتبطة بمشاريعك الأخرى | Completion rate لهذه الأوضاع، تفاعل مع المنظومة الأكبر | Medium (6-12M) |
| P6 | تشبّع السوق بألعاب سطحية + نقص المحتوى المتجدد | **AI-assisted Creator Mode** | أدوات مبسطة للاعبين لإرسال سلاسل، AI يقوم بالفحص، مجتمع مصممين | # ألغاز UGC المعتمدة، جودة التقييم | Medium (6-12M) |
| P7 | الحاجة لتدريب التفكير المنظومي | **Interdisciplinary Mode** | ألغاز تربط مفاهيم من 3 مجالات مختلفة، تظهر في Weekly Labs | Avg. completion time، تقييمات المستخدمين للتحدي | Low (بعد 12M) |

## Timeline Aligned Execution

### المرحلة 0–3 أشهر (MVP+)
1. **Daily Brain Gym / Mind Flex Score**  
   - استغلال Adaptive Learning الحالي لإظهار توصية الروابط وتغيير واجهة الصفحة الرئيسية.  
   - إضافة مؤشرات بسيطة (عدد الأيام المتتالية، متوسط الوقت لكل رابط).
2. **Family Link Sessions v1**  
   - تحسين تجربة Group Mode، إضافة هوية العائلة، تتبع “جلسات الأسبوع”، إشعارات بسيطة داخل التطبيق.

### المرحلة 3–6 أشهر
1. **Culture Link Packs + Daily World Challenge**  
   - إنتاج حزم محتوى عربية/محلية + حدث يومي موحّد.  
   - يحتاج أدوات تحكم بالمحتوى وواجهة اختيار الحزم.
2. **Classroom / Workshop Pilot**  
   - Offline bundle + تقارير بسيطة للمدرّس (CSV أو شاشة ويب بسيطة).  
   - التعاون مع مدرسة أو مركز واحد لاختبار النموذج.

### المرحلة 6–12 أشهر
1. **Life Journey Modes (Regret / Emotion / Longevity)**  
   - أوضاع قصيرة داخل Daily/Events تربط سلوك ↔ نتيجة ↔ حل.  
   - ربط النتائج بتطبيقاتك الأخرى (Insight في منصة القرارات/الندم).
2. **Creator Mode + AI Moderation**  
   - إطلاق أدوات إنشاء مبسطة مع موافقات بشرية في البداية.  
   - لاحقًا ربطه بنظام نقاط أو شارات.

### ما بعد 12 شهر
- **Interdisciplinary Mode + Macro Reasoning Labs**  
  - مزج الفيزياء/التاريخ/الاقتصاد في لغز واحد لتدريب التفكير المنظومي.  
  - يحتاج ربط أعمق مع رزم تعليمية/بحثية أكبر.
